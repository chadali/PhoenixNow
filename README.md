# PhoenixNow
This app facilitates signins and roll calls for schools that aren't especially centralized.

##Current Features

* Admin/User authentication
* Admin Portal
* User Attendance History
* SSL Encryption
* REST API / Android Mobile APP
* Hosted on a AWS EC2 Instance
